Sobre o jogo

Este jogo foi criado para auxiliar os estudantes a praticar operações básicas de matemática. O jogador deve responder o maior número possível de perguntas de adição, subtração, multiplicação e divisão em 1 minuto.

Esta é uma página de jogos de matemática desenvolvida para um trabalho escolar do curso técnico em desenvolvimento de sistemas.
